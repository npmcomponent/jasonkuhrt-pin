
# pin.css

  class-based absolute positioning

## Installation

    $ component install jasonkuhrt/pin.css

## API

  8 Available classes (16 counting aliaes):

  ```
  .pin-ne | .pin-tl
  .pin-n  | .pin-t
  .pin-nw | .pin-tr
  .pin-w  | .pin-r
  .pin-sw | .pin-br
  .pin-s  | .pin-b
  .pin-se | .pin-bl
  .pin-e  | .pin-l
  ```
